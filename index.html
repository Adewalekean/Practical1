First Heading for Git Practical

The DevMentor tutorial on git was exciting as we learnt the following

  • How to create a repo on git profile
  • How to link the repo to local using "git remote add origin 'git repo url'" code.
  • How to upload first file to created repo by;
    1. Pulling from main branch using "git pull origin main" code
    2. Creating a new branch using "git checkout -b 'branch name' code
    3. Stag files using "git add ." code with the period '.' used for stagging all the files in the local repo.
    4. Commit file using "git commit -m 'comment about the changes'" code.
    5. Push file using the "git push origin 'branch name'" code.
  • Become a contributor to files on git repo by sending and accepting request on git profile (settings).
  • How to download repo files to local using "git clone 'repo url/link'" code.
  • How to push files after modification by;
    1. Pulling from main branch using "git pull origin main" code.
    2. Creating a new branch using "git checkout -b 'branch name' code.
    3. Stag files using "git add ." code with the period '.' used for stagging all the files in the local repo.
    4. Commit file using "git commit -m 'comment about the changes'" code.
    5. Push file using the "git push origin 'branch name'" code.
